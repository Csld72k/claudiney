Main Formatting

Bold / Contrast

In this sentence we have a term in bold using the B (non-semantic) tag.

In this sentence, I have a featured term using the STRONG tag (semantics).

Italics / Emphasis

In this sentence, we have a term in italics using the I tag (non-semantic).

In this sentence, we have a term in emphasis using the EM tag (semantics).

Marked Text

We can also create a marked text using the MARK tag.

And in the other paragraph, we have other marked text at the end.

Big and Small Text

We are creating a big text and a small text in that paragraph.

Deleted Text

We can arrange a text as deleted to indicate that it should be read but not considered.

Inserted Text

We can arrange a text as inserted to add emphasis and indicate that it was added later.

There is also the underlined with the U tag (non-semantic)
