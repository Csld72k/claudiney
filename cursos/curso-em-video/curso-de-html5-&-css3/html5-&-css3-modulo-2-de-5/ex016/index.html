Colors in CSS

To understand this exercise, look very closely at the file's source code.

This title is colored using colors by name.

This one is already using colors by Hexadecimal code.

This other one is colored using RGB codes (red, green, blue).

This one here is using HSL codes (hue, saturation, light).

The resulting colors are exactly the same, but the representation format changes depending on the chosen pattern.

It is also possible to use transparency, using the RGBA format (red, green, blue, alpha).
